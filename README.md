<h4 align="center">SUB Is a tool that performs a comprehensive search on sites to find subdomains</h4>

### _Install_
```bash
$ git clone https://github.com/ymeesnacuoy/subdomain.git
$ cd subdomain
$ chmod +x indtall.sh
$ ./indtall.sh
$ sub google.com
```
